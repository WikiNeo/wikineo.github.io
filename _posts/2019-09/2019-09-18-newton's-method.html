--- title: "Newton's Method" published: true --- In numerical analysis, Newton's method is a root-finding algorithm which produces successively better approximations to the roots (or zeroes) of a real-valued function. $$ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \tag{1}\label{1} $$ We can use it to find square root of a. let $$ f(x) = x^2 - a \tag{2}\label{2} $$ then $$ f'(x) = 2x \tag{3}\label{3} $$ Apply \( \eqref{2} \) and \( \eqref{3} \) to \( \eqref{1} \) we get $$ x_{n+1} = x_n - \frac{x_n^2 - a}{2x_n} \tag{4}\label{4} $$ With \( \eqref{4} \) we get $$ x_{n + 1} = \frac{1}{2} (x_n + \frac{a}{x_n}) \tag{5}\label{5} $$ With equation \( \eqref{5} \) we can write the following code 

int mySqrt(int x) {

    auto cur = 1.0;

    while(true){
        auto pre = cur;
        cur = (cur + x/cur) / 2;
        if(abs(cur - pre) < 1e-6) {
            return cur;
        }
    }
}
